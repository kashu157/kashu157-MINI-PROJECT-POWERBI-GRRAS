# 📊 Online Course Analytics Dashboard

## Overview

This Power BI dashboard analyzes online course data to provide insights into learner engagement, course performance, language preferences, instructor ratings, subtitles, and course duration.

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Data Cleaning

## Business Questions Solved

- Total Courses
- Average Rating
- Total & Average Views
- Course Type Analysis
- Language Distribution
- Language Preferences
- Subtitle Impact
- Top Instructors
- Course Duration Analysis
